# Spotify Backend API 🎧

A Spotify-like backend API built using **Node.js, Express, and MongoDB**.  
This project allows artists to upload music, create albums, and users to fetch music and album data.

---

## 🚀 Features

- User Authentication (JWT + Cookies)
- Role-based authorization (Artist / User)
- Upload music files
- Create albums
- Get all music
- Get all albums
- Get album by ID
- MongoDB relations with populate()

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer (file upload)
- ImageKit (music storage)
